CI/CD Süreci: Jenkins & Docker

Jenkins container’ını Docker ile başlattık, build pipeline çalıştırdık.

Hatalar / Karşılaşılan Zorluklar: Docker içinde Jenkins çalıştırırken `docker` komutu bulunamadı (OCI runtime exec failed).

CI/CD Süreci: GitHub & Jenkins

Jenkins pipeline ile GitHub repository’den kod çektik ve build ettik.

CI/CD Süreci: HTML & Build

Pipeline build edildi ve deploy işlemi başlatıldı.
